# 📊 DataSentinel - Data Quality Dashboard

DataSentinel is a Streamlit-based Data Quality Dashboard that helps users analyze, validate, and monitor dataset quality. It provides an interactive interface for profiling datasets, detecting anomalies, validating data quality rules, and generating reports.

---

## 🚀 Features

- 📁 Upload CSV or Excel datasets
- 📊 Interactive Data Profiling
- ✅ Data Quality Validation
- 🔍 Missing Value Detection
- 📈 Statistical Summary
- 🚨 Anomaly Detection
- 📋 Automated Data Quality Reports
- 🔐 Secure User Authentication
- 📉 Interactive Visualizations
- 📥 Export Results

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- SQLAlchemy
- OpenPyXL
- YAML
- Bcrypt
- Streamlit-Authenticator

---


## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/DataSentinel.git
```

Move into the project directory

```bash
cd DataSentinel
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
